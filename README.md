esp8266 HTTP Sketch
=========================
Arduino IDE sketch to use HTTP GET and POST in esp8266. PHP example included.

How to use?
-------------------------
Add esp8266 board to Arduino IDE before use this sketch. Then open the sketch and modify it as you want!
